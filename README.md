# ESP32 RC Controller (Open-Source Based) https://github.com/JunOllyLi/edgetx_muffin 

## Description
This project is an ESP32-based multi-channel remote controller using joystick, buttons, switches, and potentiometer.

It communicates with a PC via USB HID and can be used with EdgeTX for drone control systems.

## Features
- Multi-channel input (Joystick, Buttons, Switches, Potentiometer)
- USB HID communication using TinyUSB
- Compatible with EdgeTX
- Real-time control

## System Overview
ESP32 → USB HID → PC → EdgeTX

## Libraries Used
- TinyUSB
- ESP-IDF
